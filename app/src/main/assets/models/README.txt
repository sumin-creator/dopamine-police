Place your GLB character model here to swap from the built-in OpenGL mascot.

Recommended filename:
- intervention_character.glb

Current behavior:
- If this file exists, the app loads it through SceneView when intervention is triggered.
- If this file is missing, the app falls back to the built-in OpenGL mascot mesh.

Model tips:
- Keep file size below 8 MB for smooth startup.
- Include one idle animation clip if possible.
- Use meter scale and centered pivot.
